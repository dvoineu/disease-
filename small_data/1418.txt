# State Fact Sheets

Minus

Related Pages

National Tobacco Control Program (NTCP) State Fact Sheets are available for
all 50 states and the District of Columbia in the map below. The fact sheets
present the latest nationally representative data available for each state in
PDF format. Click on the map below to view the fact sheets.For more
information on the data indicators in the NTCP State Fact Sheets, please
access Data Sources and Methodologies documentpdf icon[PDF-650 KB] and
References documentpdf icon[PDF-629 KB].

For questions about the NTCP State Fact Sheets, contact OSH.

For additional information on state tobacco control program funding, see
National Tobacco Control Program Funding.
