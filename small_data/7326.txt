# Evaluaciones de riesgos para la salud

English (US)

Minus

Solicite una evaluacion de riesgos para la salud

Encuentre el informe de una evaluacion de riesgos para la salud (HHE)

El Programa de Evaluacion de Riesgos para la Salud ayuda a empleados,
funcionarios sindicales y empleadores a averiguar si hay riesgos para la salud
presentes en el lugar de trabajo y recomienda formas de reducirlos y prevenir
las enfermedades ocupacionales. Las evaluaciones se realizan sin costo para
los trabajadores, funcionarios sindicales y empleadores.
